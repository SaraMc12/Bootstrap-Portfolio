About

In this project, I created a Trivia game based on one of my favorite shows “The Golden Girls” using JavaScript for the logic and jQuery to manipulate HTML and to style I did a mixture of bootstrap and CSS.

I later decided to change all of it back to vanilla JavaScript as I felt it was important to gain a clear understanding of the programming language. It is much more code, but it helped me gain more of an understanding of global and local scope and how to better create “stories” that I would test instead of writing one large block of code.

In the end I did get most of the results I wanted, enough that I felt good sharing it on my portfolio although I feel that I will probably go back and tinker with it ab it more till it is exactly how I want it.
