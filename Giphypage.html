More About Giphtastic!

In this project, I used the GIPHY API to make a dynamic web page that populates with gifs. Some of them are prechosen but then the user is able to upload new ones of their choice by calling the GIPHY API. I used JSON to call on on giffs based on a certian rating. I used JavaScript and jQuery to change the HTML and mixture of Bootstrap and CSS to style it. For bit more programming I added an automatic pause to the gif, when the user clicks on the image it unpauses and pauses when clicked again.
