More About Giphtastic!

In this project, I create a gif generator as a way to minimize boredom and maximize laziness . To get the user started I have already populated the site with some of my favorite gif suggestions, and then added a search bar so they may choose their own favorites, the site then automatically saves their search for future lazy gif enjoyment. To make this happen I used the GIPHY API and JSON to call on any gif that is enteres in the search bar. To make it more family friendly I set a paramiter of G and PG rating. To create this site I used JavaScript and jQuery and for the UI I used Bootstrap. For bit more functionality I added an automatic pause to the gif, when the user clicks on the image it unpauses and pauses when clicked again.
