More about Sweet Treats Memory Game

This is a game I made becasue I use to love playing memory as a kid and I wanted to find new fun ways to practice building games with javascript.

This game was a lot of fun to make becasue it was a great learning opportunity to get more advanced in CSS by learning how to create a transition and a 3d effect everythime the card is flipped.
