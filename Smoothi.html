More about Smothii

This was a group project where five of us had 2 weeks to create a fully functioning app with a database. We decided to do something different by creating an app for a specific machine. A Smoothie machine that would make fresh made-to-order smoothies.

We thought this would be a great idea to have in a high end gym for the health concious person on the go. This app was build for a touch screen tablet so I decided to challenge myself and build some draggable fruit buttons as a fun way for the customer to create their own smoothie.
