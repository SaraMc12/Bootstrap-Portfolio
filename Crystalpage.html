More About Space Guessing Game

This was one of the earlier projects I made in my bootcamp. I created this game as a way for me to practice dynamically updating my HTML pages using jQuery. I styled the game using CSS and gave it a retro feel, most likely since I had been watching Stranger Things and felt nostalgic for Nintendo and Atari.

In this game there are several spaceships that are each worth a different mystery value. Your objective is to choose enough spaceships to add up to the number the aliens chose. If you match the Aliens number you win! If you lose the aliens get to take you. Ready to preactice your mental math skills?
